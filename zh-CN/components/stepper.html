Stepper 步骤条

用来标识和组织完成一个任务时的信息

何时使用

当完成的目标需要分布来执行，且带有明确的先后次序

一个任务的每一步都需要较为复杂的信息和执行操作

基础用法

  • 1 账号信息
  • 2 邮箱激活
  • 3 信息登记
一个复杂任务需要拆分多个步骤完成，步骤数量不宜过多，4-7个为宜
import React from 'react'
import Stepper from '@hi-ui/hiui/es/stepper'
class Demo extends React.Component {
  render() {
    const list = [
      {
        title: '账号信息',
      },
      {
        title: '邮箱激活',
      },
      {
        title: '信息登记',
      },
    ]
    return (
      <Stepper
        data={list}
        current={1}
      />
    )
  }
}

图标用法

  • 账号信息
  • 邮箱激活
  • 信息登记
每个步骤可用图标明确表示含义，带来丰富生动的效果
import React from 'react'
import Icon from '@hi-ui/hiui/es/icon'
import Stepper from '@hi-ui/hiui/es/stepper'
class Demo extends React.Component {
  render() {
    const list = [
      {
        title: '账号信息',
        icon: <Icon name='user' />,
      },
      {
        title: '邮箱激活',
        icon: <Icon name='time' />
      },
      {
        title: '信息登记',
        icon: <Icon name='bars' />
      },
    ]
    return (
      <Stepper
        itemLayout="vertical"
        data={list}
        current={1}
      />
    )
  }
}

垂直用法

  • 1 账号信息
  • 2 邮箱激活
  • 3 信息登记
步骤与内容通过左右结构可以有效利用页面空间
import React from 'react'
import Icon from '@hi-ui/hiui/es/icon'
import Stepper from '@hi-ui/hiui/es/stepper'
class Demo extends React.Component {
  render() {
    const list = [
      {
        title: '账号信息',
        content: '请输入账号信息',
      },
      {
        title: '邮箱激活',
        content: '请输入邮箱',
      },
      {
        title: '信息登记',
        content: '请输入个人信息',
      },
    ]
    return (
      <div
        style={{height: '500px', width: '130px'}}
      >
        <Stepper
          data={list}
          current={1}
          placement='vertical'
        />
      </div>
    )
  }
}

Props

参数说明类型可选值默认值
data步骤数据项源DataItem[]--
current当前步骤位置索引，从 0 开始计数number--
placement水平或垂直展示步骤条string'vertical' | 'horizontal''horizontal'
itemLayout步骤项的布局方式string'vertical' | 'horizontal''vertical'
onChange点击切换步骤时触发(current: number) => void--

Type

DataItem

参数说明类型可选值默认值
title步骤项标题string | ReactNode--
content步骤项内容string | ReactNode--
icon步骤项 icon, 为 string 时为 HIUI Icon 组件的 namestring | ReactNode--
