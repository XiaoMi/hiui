Slider 滑块

通过简单拖拽的交互形式实现数值的设置

何时使用

设置多个参数且要求操作所即见即所得时

基础用法

10
滑动输入连续或离散数据的单点值或范围值
import React from 'react'
import Slider from '@hi-ui/hiui/es/slider'


class Demo extends React.Component {


  constructor() {
    super()
    this.state = {
      value: 10
    }
  }


  onChange(v){
    // console.log(v)
    this.setState({
      value:v
    })
  }
  
  render() {
    const {value} = this.state
    return (
      <Slider value={value}  onChange={(value)=>this.onChange(value)}/>
    )
  }
}

输入框用法

10
1090
滑动输入连续或离散数据的单点值或范围值
import React from 'react'
import { Slider, Input, Grid } from '@hi-ui/hiui'
class Demo extends React.Component {


  constructor() {
    super()
    this.state = {
      value: 9,
      valueCache:10,
      max:90,
      min:10
    }
  }
  componentDidMount(){
    const {min,max,value} = this.state
    if(value<min){
      this.setState({
        value:min
      })
    }
    if(value>max){
      this.setState({
        value:max
      })
    }
  }
  onChange(e){
    let valueCache = e.target.value.replace(/[^0-9]/g, '')
    this.setState({
      valueCache
    })
  }
  
  handleInput(value){
    const {min,max} = this.state
    if(value<min){
      value = min
    }
    if(value>max){
      value = max
    }
    this.setState({
      valueCache:value,
      value
    });
  }


  render() {
    const {Row, Col} = Grid
    const {value,max,min,valueCache} = this.state


    return (
      <Row>
        <Col span={22}>
          <Slider defaultValue={10} onChange={(value)=>this.handleInput(value)} value={this.state.value}  max={90}  min={10}  showRangeLabel={true}/>
        </Col>
        <Col span={2}>
          <Input    
            style={{width:80,marginLeft:30}}
            autoFocus
            value={valueCache}
            onChange ={(e)=>this.onChange(e)}
            onBlur={(e)=>this.handleInput(+e.target.value)}
            onKeyDown={(e)=>{
              if(e.keyCode == 13){
                this.handleInput(+e.target.value)
              }
            }}
          />
        </Col>
      </Row>
    
    )
  }
}

可自定义提示内容的格式
import React from 'react'
import Slider from '@hi-ui/hiui/es/slider'


class Demo extends React.Component {


  constructor() {
    super()
    this.state = {
      value: 80
    }
  }


  formatter(value){
    console.log(value)
    return value+'英寸'
  }


  render() {
    return (
      <Slider defaultValue={this.state.value}   tipFormatter={this.formatter}/>
    )
  }
}

可选颜色

80
可选滑块条的颜色
import React from 'react'
import Slider from '@hi-ui/hiui/es/slider'


class Demo extends React.Component {
    constructor() {
        super()
        this.state = {
            value: 80,
            color: [{
                color: '#4285f4',
                type: 'primary'
            }, {
                color: '#E65C2F',
                type: 'danger'
            }, {
                color: '#1da653',
                type: 'success'
            }, {
                color: '#e19d0c',
                type: 'warning'
            }],
            type: 'primary'
        }
    }
    onHandleChange(type) {
        this.setState({
            type
        })
    }
    render() {
        const { value, color, type } = this.state
        return (
            <>
                <div style={{ display: 'flex', 'justifyContent': 'flex-end' }}>
                    {color.map(item => <span style={{
                        width: 25, 
                        height: 22, 
                        marginRight: 12, 
                        background: item.color, 
                        display: 'inline-block', 
                        cursor: 'pointer', 
                        boxShadow: type === item.type ? '0px 2px 4px 0px rgba(0,0,0,0.5)' : 'none' }} 
                        onClick={() => { this.onHandleChange(item.type) }} key={item.color}></span>)}
                </div>
                <Slider defaultValue={this.state.value} type={type} />
            </>
        )
    }
}

自定义步长

30
0°C26°C37°C100°C
按定义步长输入离散型数值，可加入特殊位置
import React from 'react'
import Slider from '@hi-ui/hiui/es/slider'


class Demo extends React.Component {


  constructor() {
    super()
    this.state = {
      value: 30,
      marks:{
        0: '0°C',
        26: '26°C',
        37: '37°C',
        100: '100°C',
      }
    }
  }


  render() {
    const {value,marks} = this.state
    return (
      <Slider defaultValue={value} marks={marks} step={9}/>
    )
  }
}

Props

参数说明类型可选值默认值
type类型string'primary' | 'danger' | 'success' | 'warning''primary'
defaultValue设置初始取值number-0
value设置当前取值number-0
min最小值number-0
max最大值number-100
disabled值为 true 时，滑块为禁用状态booleantrue | falsefalse
tipFormatter自定义 Tooltip 中显示文案value => ReactNode--
marks刻度标记,key 的类型必须为 number 且取值在闭区间 [min, max] 内object-{}
step步长number-1
vertical值为 true 时，Slider 为垂直方向booleantrue | falsefalse
showRangeLabel是否显示范围labelbooleantrue | falsefalse
