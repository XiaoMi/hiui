Badge 红点 / 徽标

红点用来展示新消息的提示，徽标用于展示消息的数量、提示

何时使用

有新消息提示时

常见于图标、文字标题、按钮等位置

红点

标识是否有新消息
import React from 'react'
import Button from '@hi-ui/hiui/es/button'
import Badge from '@hi-ui/hiui/es/badge'
class Demo extends React.Component {
  render () {
    return (
      <div>
        <Badge type='dot'>
          <Button type='default'>最新报表</Button>
        </Badge>
      </div>
    )
  }
}

徽标

8
标识消息数或简洁描述
import React from 'react'
import Button from '@hi-ui/hiui/es/button'
import Badge from '@hi-ui/hiui/es/badge'
class Demo extends React.Component {
  render () {
    return (
      <Badge content={8}>
        <Button type='default'>最新报表</Button>
      </Badge>
    )
  }
}

参数说明类型可选值默认值
content气泡显示内容string | number--
type气泡显示的形态，可选带文字的气泡或小红点string'bubble' | 'dot''bubble'
max气泡显示的最大值，超过值用'+'号替代number-99
visible是否显示气泡booleantrue | falsetrue
color气泡颜色string--
offset设置状态点的位置偏移[left:number, top:number]--
