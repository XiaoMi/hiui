Card 卡片

用来展示不同类型信息或信息组的轻量容器

何时使用

当一组同类信息不能自然形成区块感时，需借助卡片来组织

当一个应用或功能由大量信息组成，需要通过重点信息快速识别时

后台展示概览信息时

内容

卡片可以展示多种类型的信息：

  • 文本、数字
  • 段落
  • 信息图表，如环形图、柱状图、表格、折线图等
  • 图片、图片组
  • 图标
  • 列表

基础用法

标题

基础卡片

用卡片封装独立的信息实体，如模块、功能、项目、应用等，或还原实际生活中的卡片应用
import React from 'react'
import Card from '@hi-ui/hiui/es/card'
class Demo extends React.Component {
  render() {
    return (
      <Card title="标题" size='default' >
        <p>基础卡片</p>
      </Card>
    )
  }
}
