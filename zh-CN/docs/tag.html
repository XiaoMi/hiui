Tag 标签

基本用法

待审批
审批中
已通过
未通过
面型直角标签 重要提示所处状态，如“已通过”、“审批中”
import Tag from '@hi-ui/hiui/es/tag'
import React from 'react'
class Demo extends React.Component {
  constructor() {
    super()
    
  }
  render () {
    return (
      <div>
        <Tag type="warning" onClick={()=>{console.log('待审批')}}>待审批</Tag>
        <Tag>审批中</Tag>
        <Tag type="success">已通过</Tag>
        <Tag type="danger">未通过</Tag>
      </div>
    )
  }
}

线型标签

原创
活动
聚焦
最新
线型直角标签 一般提示内容，如“原创”、“活动”
import Tag from '@hi-ui/hiui/es/tag'
import React from 'react'
class Demo extends React.Component {
  constructor() {
    super()
    
  }
  render () {
    return (
      <div>
        <Tag type="warning" appearance="line">原创</Tag>
        <Tag appearance="line">活动</Tag>
        <Tag type="success" appearance="line">聚焦</Tag>
        <Tag type="danger" appearance="line">最新</Tag>
      </div>
    )
  }
}
