Stepper 步骤

用来标识和组织完成一个任务时的信息

何时使用

  • 当完成的目标需要分布来执行，且带有明确的先后次序
  • 一个任务的每一步都需要较为复杂的信息和执行操作

基础用法

  • 1 账号信息
  • 2 邮箱激活
  • 3 信息登记
import React from 'react'
import Stepper from '@hi-ui/hiui/es/stepper'
class Demo extends React.Component {
  render() {
    const list = [
      {
        title: '账号信息',
      },
      {
        title: '邮箱激活',
      },
      {
        title: '信息登记',
      },
    ]
    return (
      <Stepper
        data={list}
        current={1}
      />
    )
  }
}

  • 账号信息

      },
    ]
    return (
      <Stepper
        data={list}
        current={1}
      />
    )
  }
}

上下结构

  • 1 账号信息请输入账号信息
  • 2 邮箱激活请输入邮箱
  • 3 信息登记请输入个人信息
上下结构只需加入 itemLayout="vertical"；只有上下结构有 text 描述
import React from 'react'
import Icon from '@hi-ui/hiui/es/icon'
import Stepper from '@hi-ui/hiui/es/stepper'
class Demo extends React.Component {
  render() {
    const list = [
      {
        title: '账号信息',
        content: '请输入账号信息',
      },
      {
        title: '邮箱激活',
        content: '请输入邮箱',
      },
      {
        title: '信息登记',
        content: '请输入个人信息',
      },
    ]
    return (
      <Stepper
        data={list}
        current={2}
        itemLayout='vertical'
      />
    )
  }
}

Props

参数说明类型可选值默认值
data步骤数据项源DataItem[]--
current当前步骤位置索引，从 0 开始计数number--
placement水平或垂直展示步骤条string'vertical' | 'horizontal''horizontal'
itemLayout步骤项的布局方式string'vertical' | 'horizontal''vertical'

Type

DataItem

参数说明类型可选值默认值
