Radio 单选按钮

单选按钮是用来在一组平铺选项中选择目标选项，每次只选一个

何时使用

  • 需要快速完成多选一且备选项数量不多时
  • 常见于后台设置、调研问卷等场景

普通模式

import React from 'react'
import Radio from '@hi-ui/hiui/es/radio'
class Demo extends React.Component {
  render() {
    return (
      <Radio.Group
        data={['手机类', '电脑类', '生活类', '其它']}
        defaultValue='手机类'
        onChange={(data) => console.log(data)}
      />
    )
  }
}

按钮模式

import React from 'react'
import Radio from '@hi-ui/hiui/es/radio'
class Demo extends React.Component {
  constructor() {
    super()
    this.list = [{
      id: 0,
      content: '手机类'
    }, {
      id: 1,
      content: '电脑类',
      disabled: true
    }, {
      id: 2,
      content: '生活类'
    }, {
      id: 3,
      content: '其它'
    }]
  }
  render() {
    return (
      <Radio.Group
        data={this.list}
        type='button'
        defaultValue={0}
        onChange={(data) => console.log(data)}
      />
    )
  }
}
