# fields

- basic 基础组件，通常是表单组件，直接来自 HiUI 的组件
- enhance 增强组件，基础组件之上的包装组合
- semantic 语义组件，基础组件之上的语义包装，通常包含与语义相关的只读或可编辑的多块逻辑
