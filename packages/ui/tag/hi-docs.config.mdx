## Type

### TagGroupDataItem

| 参数     | 说明                                   | 类型      | 可选值        | 默认值 |
| -------- | -------------------------------------- | --------- | ------------- | ------ |
| id       | 标签唯一 id                            | ReactText | -             | -      |
| title    | 标题                                   | ReactNode | -             | -      |
| closable | 标签是否可以关闭                       | boolean   | true \| false | false  |
| editable | 标签是否可以修改                       | boolean   | true \| false | false  |
| maxWidth | 最大宽度，如超出，则截断末尾添加省略号 | number    | -             | -      |
