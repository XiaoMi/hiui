# Transfer 穿梭框

在一定展示空间里对选项进行单个或批量移动从而完成挑选的数据容器。

## 何时使用

一组数据进行两种状态的分类时

有更多的空间进行选择时

需快速批量的完成选项归类且可调整顺序时

### TransferDataItem

| 参数     | 说明            | 类型      | 可选值        | 默认值 |
