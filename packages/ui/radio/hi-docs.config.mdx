# Radio 单选

单选是用来在一组平铺选项中选择目标选项，每次只选一个。

## 何时使用

需要快速完成多选一且备选项数量不多时

常见于后台设置、调研问卷等场景

## Type

### RadioDataItem

| 参数     | 说明            | 类型      | 可选值        | 默认值 |
| -------- | --------------- | --------- | ------------- | ------ |
| id       | 选项唯一标识 id | ReactText | -             | -      |
| title    | 选项显示内容    | ReactNode | -             | -      |
| disabled | 是否禁用该选项  | boolean   | true \| false | false  |
