## Type

### UploadFileItem

| 参数        | 说明         | 类型   | 可选值                              | 默认值 |
| ----------- | ------------ | ------ | ----------------------------------- | ------ |
| fileId      | 上传文件 id  | string | -                                   | -      |
| fileType    | 文件类型     | string | -                                   | -      |
| name        | 文件名       | string | -                                   | -      |
| uploadState | 上传文件状态 | string | 'success' \| 'loading' \| 'error' | -      |
| url         | 上传文件地址 | string | -                                   | -      |
