# Scrollbar 滚动条

滚动条用于跨系统统一滚动视觉样式、行为，便捷监听滚动事件。

## 何时使用

需要统一滚动条视觉样式，或者需要监听某些特殊滚动事件，比如滚动到底部、滚动到顶部。
