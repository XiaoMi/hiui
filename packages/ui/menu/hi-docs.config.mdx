## Type

### MenuDataItem

| 参数     | 说明                                             | 类型           | 可选值        | 默认值 |
| -------- | ------------------------------------------------ | -------------- | ------------- | ------ |
| id       | 菜单项唯一标识                                   | ReactText      | -             | -      |
| title    | 菜单项标题                                       | ReactNode      | -             | -      |
| icon     | 菜单项 icon                                     | ReactNode      | -             | -      |
| disabled | 菜单项是否禁止点击                               | boolean        | true \| false | false  |
| children | 子菜单项配置                                     | MenuDataItem[] | -             | -      |
