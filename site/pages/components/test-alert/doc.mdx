# Card 卡片

卡片式用来展示不同类型信息或信息组的轻量

## 何时使用

- 当一组同类信息不能自然形成区块感时，需借助卡片来组织。
- 当一个应用或功能由大量信息组成，需要通过重点信息快速识别时。
- 后台展示概览信息时。

## 信息构成

卡片可以展示多种类型的信息：

- 文本、数字
- 段落
- 信息图表、如环形图、柱状图、表格、折线图等
- 图片、图片组
- 图标
- 列表

## Card Attributes

| 参数        | 说明                                                                                                    | 类型                     | 可选值                   | 默认值 |
| ----------- | ------------------------------------------------------------------------------------------------------- | ------------------------ | ------------------------ | ------ |
| title       | 卡片标题                                                                                                | String \| Element        | -                        | -      |
| size        | 卡片大小<br/> small -> 276px <br/> middle -> 376px <br/> large -> 576px <br/> 不传入 size 将占 100%宽度 | String                   | small \| middle \| large | 100%   |
| type        | 卡片类型                                                                                                | String                   | simple                   | -      |
| style       | 自定义样式                                                                                              | Object                   | -                        | -      |
| hoverable   | 鼠标移入卡片是否显示浮起效果                                                                            | Boolean                  | true \| false            | false  |
| extra       | 卡片右上的扩展按钮                                                                                      | Array[Elemnt] \| Element | -                        | -      |
| extraShow   | 扩展按钮的显示模式                                                                                      | String                   | stay \| hover            | stay   |
| cover       | 图片卡片                                                                                                | ImgElement               | -                        | -      |
| description | 图片卡片的信息描述                                                                                      | String \| Element        | -                        | -      |
| disabled    | 是否禁用卡片                                                                                            | Boolean                  | true \| false            | -      |
