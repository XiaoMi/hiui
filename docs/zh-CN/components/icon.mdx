## Icon Attributes

| 参数      | 说明         | 类型   | 可选值       | 默认值 |
| --------- | ------------ | ------ | ------------ | ------ |
| name      | 图标名称     | string | 参考图标集合 | -      |
| className | 自定义 class | string | -            | -      |
