# Stepper 步骤

用来标识和组织完成一个任务时的信息

## 何时使用

- 当完成的目标需要分布来执行，且带有明确的先后次序
- 一个任务的每一步都需要较为复杂的信息和执行操作

## Stepper Attributes

| 参数      | 说明                            | 类型              | 可选值        | 默认值 |
| --------- | ------------------------------- | ----------------- | ------------- | ------ |
| id        | 组件 id                         | String            | -             | -      |
| className | 样式名                          | String            | -             | -      |
| list      | 步骤元素列表                    | 参见 List Options | -             | -      |
| current   | 当前步骤位置索引，从 0 开始计数 | Number            | -             | -      |
| vertical  | 是否竖直显示                    | Boolean           | true \| false | false  |
| up        | 在水平显示时符号是否在上        | Boolean           | true \| false | false  |

## List Options

| 参数  | 说明        | 类型              | 可选值 | 默认值 |
| ----- | ----------- | ----------------- | ------ | ------ |
| title | 显示文案    | Element \| String | 必需   | -      |
| text  | 副文本      | Element \| String | -      | -      |
| icon  | 自定义 icon | Element           | -      | -      |

## Props

| 参数       | 说明                            | 类型       | 可选值                     | 默认值       |
| ---------- | ------------------------------- | ---------- | -------------------------- | ------------ |
| data       | 步骤数据项源                    | DataItem[] | -                          | -            |
| current    | 当前步骤位置索引，从 0 开始计数 | number     | -                          | -            |
| placement  | 水平或垂直展示步骤条            | string     | 'vertical' \| 'horizontal' | 'horizontal' |
| itemLayout | 步骤项的布局方式                | string     | 'vertical' \| 'horizontal' | 'vertical'   |

## Type

**_DateItem_**

| 参数    | 说明       | 类型                | 可选值 | 默认值 |
| ------- | ---------- | ------------------- | ------ | ------ |
| title   | 步骤项标题 | string \| ReactNode | -      | -      |
| content | 步骤项内容 | string \| ReactNode | -      | -      |
| icon    | 步骤项图标 | string \| ReactNode | -      | -      |
