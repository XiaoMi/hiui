# Button 按钮

常用的操作按钮。

## 标准按钮

- 突出按钮：用于较重要的功能，需要强烈引导用户点击的功能操作。
- 普通按钮：用于不重要的功能，不需要太引导用户点击的功能操作。
- 默认按钮：用于不重要的功能，不需要太引导用户点击的功能操作。
- 确定/取消按钮：适用于表单、弹窗场景。

## Props

| 参数    | 说明             | 类型                    | 可选值                                                    | 默认值    |
| ------- | ---------------- | ----------------------- | --------------------------------------------------------- | --------- |
| type    | 设置按钮类型     | string                  | 'primary' \| 'line' \| 'success' \| 'danger' \| 'default' | 'default' |
| size    | 设置按钮尺寸     | string                  | 'large' \| 'default' \| 'small'                           | 'default' |
| icon    | 设置按钮图标     | string                  | 详见 [`<Icon/>`](/hiui/zh-CN/docs/icon) 组件 ︎            | -         |
| onClick | 点击按钮时的回调 | (e: MouseEvent) => void | -                                                         | -         |
