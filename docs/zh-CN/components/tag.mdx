## Props

| 参数    | 说明             | 类型                    | 可选值                                                    | 默认值    |
| ------- | ---------------- | ----------------------- | --------------------------------------------------------- | --------- |
| type    | 设置状态     | string                  | 'primary' \| 'success' \| 'warning' \| 'danger' | 'primary' |
| appearance    | 设置样式类型     | string                  | 'default' \| 'line' | 'default'  |
| onClick | 点击按钮时的回调 | (e: MouseEvent) => void | -                                                         | -         |
