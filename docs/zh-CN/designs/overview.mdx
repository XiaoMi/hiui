# 概览

设计模式是从宏观的角度对信息结构、页面关系和界面元素间关系进行定义的抽象框架，可对设计师在设计过程中做出指引，同时提升团队对设计的整体认知，从而节约沟通成本，有序推进项目进度。

HIUI 的设计模式，基于对现有业务系统的环境和现状和HIUI自身的定位而生成的，目的是形成业务系统的统一性。
