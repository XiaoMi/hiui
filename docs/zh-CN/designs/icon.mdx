# 图标

图标是 UI 设计中必不可少的组成。一般图标的设计是将某个抽象的概念转化为清晰易读的图形，从而降低用户的理解成本，提升界面的美观度。在产品设计体系中实现有一套美观、一致、易用、便于延展的图标体系是任何一个设计系统必须努力做到的。我们相信一整套优质的图标对于设计质量的影响是非常巨大的。

## 设计原则

- 清晰：设计表意清晰的图标，不对用户的认知造成困扰。
- 简单：在表意清晰准确的基础上，尽量保持图形的简洁，不做过多装饰。
- 幸福：赋予图标适度的感情。

### 圆角

圆角的规格采用 8 的倍数原则，分别为 32 64，图标内部空间的圆角为直角。

### 线

线的规格采用 8 的倍数原则，图标线的宽度统一为 64。

## 功能图标

功能图标作为特殊图标，一般用于首页的功能入口，图标与基础图标不同，活泼、鲜明、清晰是功能图标的特点。
