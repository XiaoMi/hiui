# 布局

布局是对信息在页面分布的空间规划，友好的布局可以展示给用户清晰的信息结构。目前 HIUI 定义了三种布局。

## 上下结构-顶部导航-定宽居中

适用于门户类网站，一级导航数量在 5-7 个，可支持 3 级以上信息层级。

<img src="<BASE_URL>/static/images/designs/layout-1.png" srcSet="<BASE_URL>/static/images/designs/layout-1@2x.png 2x" alt="上下结构-顶部导航-定宽居中" />

## 左右结构-侧导-通栏

适用于信息系统或应用系统，信息层级较多，业务复杂，可支持 4 级以上的信息层级。
