# 概述

## 为什么做 HIUI

小米信息部的中后台信息系统数量百余个，业务范围覆盖物流、销售、企业办公、数据、供应链等多个领域，而每个系统来源于相互独立的项目团队，在视觉表现和交互行为上呈现出多样化现象，不仅带来了成倍增长的人力成本，低效的项目效率，也导致用户的学习门槛高，培训成本变高，用户体验却不尽人意。

为此，我们成立了专业的项目团队打造了 HIUI，致力提供丰富的界面组件和模板满足复杂多样的业务需求，HIUI 的出现带来了以下优势。

- 有效释放前端开发资源，缩短项目的开发周期
- 提供丰富的组件库和模板库，提高产品经理的设计原型效率
- 形成有统一视觉风格和交互一致的系列化的中后台系统
- 拥有美好、高效的用户体验

## 认识 HIUI

HIUI 是一个以用户体验一致性为出发点，为研发工程师提供类型全面的组件库和页面模板的 UI 框架。
