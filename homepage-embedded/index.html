使用方法

只需3步，快速体验

第一步

在Agent中新建项目

pnpm create vite my-app --template react cd my-app pnpm add @hi-ui/core @hi-ui/hiui
第二步

全局安装skill

git clone --depth 1 git@github.com:XiaoMi/hiui.git cd hiui  bash skills/hiui-page-workflow/install.sh 把hiui design接入到当前项目中
第三步

对AI说出你的需求

@hiui-page-workflow 帮我做一个订单管理页，细化需求生成页面并验收
