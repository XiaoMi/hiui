# Open Source Sync Policy

Source: `<path-to-global-hiui-design>`
Target: `<path-to-public-repo>`
Generated by: `scripts/sync-open-source-package.mjs`

## Included

- `SKILL.md`
- `agents/**`
- `archetypes/**`
- `docs/**` public rule and generation docs
- `examples/host-integration/**`
- `manifests/**`
- `reference/host-integration/**`
- `rules/**`
- `schemas/**`
- selected `scripts/**`, including shipped machine-public JSON contract fixtures
- selected `templates/**`
- `packages/typical-page-shells/**`
- `vendor/typical-page-shells-package.json`
- `vendor/hiui-design-typical-page-shells-*.tgz`

## Excluded

- env files and local release config
- private onboarding docs for telemetry, one-click private setup, and internal global sync
- business-line overlays and examples
- generated release outputs
- private publishing, usage telemetry, global sync, and launch-agent scripts

## Sanitization

The sync replaces known private URLs, Git remotes, local filesystem paths, Figma file keys, and Figma node IDs with placeholders. Sanitization is a guardrail, not a substitute for manual review.
