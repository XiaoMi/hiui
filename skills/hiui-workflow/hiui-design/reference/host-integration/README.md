# Host Integration Reference

This directory keeps the reference-only host integration assets that can be synced into downstream projects.

- Read `src/README.md` for the lightweight install flow and host integration rules.
- Read `src/CHECKLIST.md` for the acceptance checklist before using the synced example pages as a business-page baseline.

The default downstream page-generation surface remains:

- `typical-page:init`
- `typical-page:generate`
- `typical-page:analytics`
- `typical-page:verify`
- `typical-page:verify:visual`
